- If there are formulas in the reference questions, use LaTeX to represent them.

- language: the output language code (e.g., "es", "en").
- blueprint: a JSON object describing subject, tags, type mix, difficulty frequencies.

{

      | { "type": "essay", "prompt": string, "answer": string, "difficulty"?: "easy"|"medium"|"hard", "rationale"?: string }
    >
  }
}
