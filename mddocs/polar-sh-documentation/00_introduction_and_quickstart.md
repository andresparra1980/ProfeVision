# Introduction to Polar

Turn your software into a business with Polar. Sell digital products, subscriptions, and more without the hassle of traditional payment systems.

## Core Value Proposition

- **Merchant of Record**: Polar handles international tax compliance (VAT, GST, Sales Tax), calculating, collecting, and remitting taxes worldwide.
- **Complete Billing Infrastructure**: Subscriptions, product catalogs, customer portals, and payment flows out-of-the-box.
- **Automated Benefits**: Built-in delivery for license keys, file downloads, GitHub repo access, and Discord roles.
- **Lower Fees**: 4% + 40¢ per successful transaction with no monthly fixed costs.

## Quick Start

1. **Create Account**: Sign up at polar.sh.
2. **Create Product**: Choose between one-time or subscription, set pricing, and configure benefits.
3. **Integrate**: Use Checkout Links (No-code), Embedded Checkout, or the API.
4. **Webhooks**: Set up webhooks to sync with your database.

## Integration Options

- **Framework Adapters**: Next.js, Laravel, Nuxt, Remix, etc.
- **Native SDKs**: JS/TS, Python, Go, PHP.
- **No-Code**: Zapier, Checkout Links.

## Fees

- **Transaction Fee**: 4% + 40¢ per successful transaction.
- **Additional Fees**: +1.5% for international cards, +0.5% for subscriptions.
- **Payout Fees**: Passed through from Stripe (e.g., $0.25 + 0.25% per payout).
- **Disputes**: $15 per dispute (non-refundable).

## Support

- **Email**: support@polar.sh
- **Discord**: Community support.
- **GitHub**: For bug reports and feature requests.

## Migration

Polar offers a CLI tool `polar-migrate` to migrate products, license keys, files, and customers from Lemon Squeezy.
