## Assistant de configuration

L'assistant vous guide à travers 6 étapes pour configurer votre compte. Cela prend environ 2 minutes.

### Étape 1 : Bienvenue

Écran d'introduction. Cliquez sur **Démarrer**.

![Welcome step](/images/getting-started/onboarding-01-welcome-step.webp)

### Étape 2 : Créer l'établissement

Saisissez les informations de votre établissement :

- **Nom** : Le nom de votre école ou établissement
- **Type** : École, université, institut, académie, ou autre

![Create institution](/images/getting-started/onboarding-02-institution-step.webp)

### Étape 3 : Créer une matière

Créez votre première matière :

- **Établissement** : Sélectionné automatiquement
- **Nom** : Nom de la matière
- **Description** : Facultatif

![Create subject](/images/getting-started/onboarding-03-subject-step.webp)

### Étape 4 : Créer un groupe

Créez votre premier groupe d'étudiants :

- **Matière** : Sélectionné automatiquement
- **Nom** : Nom du groupe (par ex., "3ème A")
- **Année** : Année scolaire (facultatif)
- **Période** : Période scolaire (facultatif)

![Create group](/images/getting-started/onboarding-04-group-step.webp)

### Étape 5 : Ajouter des étudiants

Ajoutez des étudiants à votre groupe :

- **Manuel** : Saisissez le nom et l'identifiant un par un
- **Importer** : Téléchargez un fichier Excel

![Add students](/images/getting-started/onboarding-05-students-step.webp)

### Étape 6 : Terminer

Configuration terminée ! Vous verrez les prochaines étapes recommandées.

![Complete](/images/getting-started/onboarding-06-complete-step.webp)

<Callout type="info">
Votre progression est enregistrée automatiquement. Si vous quittez, vous reprendrez là où vous vous étiez arrêté.
</Callout>
