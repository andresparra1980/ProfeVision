## Primeiro acesso

Após verificar seu e-mail, entre em sua conta para começar a usar o ProfeVision.

### Login

1. Vá para a página de login
2. Insira seu e-mail e senha
3. Complete o CAPTCHA de segurança
4. Clique em **Sign in** (Entrar)

![Login form](/images/getting-started/first-login-01-welcome-modal.webp)

### Janela de boas-vindas

Em seu primeiro acesso, você verá uma janela de boas-vindas com as opções de plano:

- **Plano Gratuito**: Recursos básicos com limites
- **Plano Plus**: Recursos ilimitados


Você pode escolher:
- **Começar gratuitamente**: Continuar com o plano gratuito
- **Atualizar para o Plus**: Desbloquear todos os recursos

<Callout type="info">
Você pode atualizar para o Plus a qualquer momento nas configurações.
</Callout>

Após a janela de boas-vindas, o assistente de configuração o guiará passo a passo.
