## Première connexion

Après avoir vérifié votre e-mail, connectez-vous pour commencer à utiliser ProfeVision.

### Se connecter

1. Accédez à la page de connexion
2. Saisissez votre e-mail et votre mot de passe
3. Complétez le CAPTCHA de sécurité
4. Cliquez sur **Se connecter**

![Login form](/images/getting-started/first-login-01-welcome-modal.webp)

### Fenêtre de bienvenue

Lors de votre première connexion, vous verrez une fenêtre de bienvenue affichant les options d'abonnement :

- **Plan Gratuit** : Fonctionnalités de base avec des limites
- **Plan Plus** : Fonctionnalités illimitées

Vous pouvez choisir :
- **Commencer gratuitement** : Continuer avec le plan gratuit
- **Passer au Plan Plus** : Débloquer toutes les fonctionnalités

<Callout type="info">
Vous pouvez passer au Plan Plus à tout moment depuis les paramètres.
</Callout>

Après la fenêtre de bienvenue, l'assistant de configuration vous guidera étape par étape.
