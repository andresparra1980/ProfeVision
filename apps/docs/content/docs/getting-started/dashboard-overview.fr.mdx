## Aperçu du tableau de bord

Le tableau de bord est votre centre d'opérations dans ProfeVision.

### Barre latérale

La barre latérale gauche contient la navigation principale :

- **Tableau de bord** : Vue d'ensemble et statistiques
- **Institutions** : Gérez vos établissements d'enseignement
- **Matières** : Gérez vos matières
- **Groupes** : Organisez vos groupes d'étudiants
- **Examens** : Créez et gérez des examens
- **Étudiants** : Gérez les étudiants
- **Mon forfait** : Consultez votre abonnement et vos limites
- **Paramètres** : Paramètres du compte

![Sidebar](/images/getting-started/dashboard-02-sidebar.webp)

### Statistiques principales

En haut, vous verrez 4 cartes avec :

- Total des institutions
- Total des matières
- Groupes actifs (et archivés)
- Total des étudiants

### Section de notation

Affiche la progression de votre notation :

- Examens notés
- Temps gagné
- Efficacité

### Utilisation du forfait

Indique votre consommation actuelle :

- Générations IA utilisées
- Examens numérisés utilisés
- Jours avant la réinitialisation du cycle

### Examens récents

Liste de vos 5 derniers examens avec un accès rapide pour modifier ou consulter les résultats.

![Full dashboard](/images/getting-started/dashboard-01-overview.webp)

### Bouton de numérisation

Le bouton flottant dans le coin inférieur droit vous permet de numériser rapidement des examens.

<Callout type="info">
Vous pouvez réduire la barre latérale en cliquant sur l'icône de panneau en haut.
</Callout>
