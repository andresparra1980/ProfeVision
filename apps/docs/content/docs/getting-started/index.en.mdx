# Getting started

This guide will help you get started with ProfeVision quickly.

## Create an account

1. Go to [profevision.com](https://profevision.com)
2. Click on "Sign up"
3. Fill out the form with your details
4. Verify your email

## Create your first exam

Once you're logged in:

1. Go to the **Dashboard**
2. Click on **Create exam**
3. Add your exam questions
4. Configure grading options
5. Save and generate answer sheets

## Next step

Continue with the [scanning guide](/docs/getting-started/scanning) to learn how to grade exams automatically.
