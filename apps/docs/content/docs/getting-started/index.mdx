# Primeros pasos

Esta guia te ayudara a comenzar con ProfeVision rapidamente.

## Crear una cuenta

1. Ve a [profevision.com](https://profevision.com)
2. Haz clic en "Registrarse"
3. Completa el formulario con tus datos
4. Verifica tu correo electronico

## Crear tu primer examen

Una vez que hayas iniciado sesion:

1. Ve al **Dashboard**
2. Haz clic en **Crear examen**
3. Agrega las preguntas de tu examen
4. Configura las opciones de calificacion
5. Guarda y genera las hojas de respuesta

## Siguiente paso

Continua con la [guia de escaneo](/docs/getting-started/scanning) para aprender a calificar examenes automaticamente.
