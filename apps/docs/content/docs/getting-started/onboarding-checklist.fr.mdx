## Liste des tâches initiales

Après l'assistant de configuration, une liste flottante apparaît pour vous guider dans la réalisation de votre premier examen.

### Emplacement

- **Ordinateur** : Barre latérale droite (extensible/repliable)
- **Mobile** : Barre inférieure (au-dessus du bouton de numérisation)

![Floating checklist](/images/getting-started/checklist-01-floating.webp)

### Les 4 tâches

Complétez ces tâches dans l'ordre :

| # | Tâche | Action |
|---|-------|--------|
| 1 | **Créer un examen** | Importer un document ou utiliser le chat IA |
| 2 | **Publier l'examen** | Changer le statut de brouillon à publié |
| 3 | **Exporter le PDF** | Télécharger les feuilles de réponses |
| 4 | **Première numérisation** | Numériser une feuille avec la caméra |

![Checklist tasks](/images/getting-started/checklist-02-tasks.webp)

### Progression

- Une barre de progression indique votre avancement
- Les tâches terminées apparaissent barrées avec un ✓
- Vous ne pouvez effectuer que la tâche suivante dans l'ordre

### Créer votre premier examen

En cliquant sur « Créer un examen » :

1. Un panneau s'ouvre avec plusieurs options
2. **Importer PDF/Word** : Télécharger un examen existant
3. **Créer avec l'IA** : Discuter avec l'IA pour le générer

![Creation options](/images/getting-started/checklist-03-create-drawer.webp)

### Masquer la liste de tâches

- Cliquez sur le **X** pour la réduire
- Elle réapparaît lors du changement de page jusqu'à ce que les 3 premières tâches soient terminées
- Elle disparaît définitivement lorsque toutes les tâches sont accomplies

<Callout type="info">
Ce guide vous accompagne de zéro jusqu'à la notation de votre premier examen en quelques minutes.
</Callout>
