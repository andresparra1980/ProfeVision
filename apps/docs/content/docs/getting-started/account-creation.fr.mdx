## Créer un compte

Pour utiliser ProfeVision, vous devez créer un compte gratuit.

### Étapes

1. Allez sur la page d'inscription
2. Remplissez le formulaire :
   - **Prénom** : Votre prénom (min. 2 caractères)
   - **Nom** : Votre nom (min. 2 caractères)
   - **E-mail** : Votre adresse e-mail
   - **Mot de passe** : Minimum 6 caractères
   - **Confirmer le mot de passe** : Répétez votre mot de passe
3. Complétez le CAPTCHA de sécurité
4. Cliquez sur **Créer un compte**

![Registration form](/images/getting-started/account-creation-01-register-form.webp)

### Vérifier votre e-mail

Après l'inscription :

1. Consultez votre boîte de réception
2. Recherchez l'e-mail envoyé par ProfeVision
3. Cliquez sur le lien de vérification

<Callout type="info">
Si vous ne trouvez pas l'e-mail, vérifiez votre dossier de courriers indésirables (spams).
</Callout>

![Verification page](/images/getting-started/account-creation-02-verify-email.webp)

### Confirmation

Après avoir cliqué sur le lien, vous verrez une page de confirmation. Votre compte est prêt.

![Email confirmed](/images/getting-started/account-creation-03-email-confirmed.webp)
