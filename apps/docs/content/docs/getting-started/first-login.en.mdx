## First login

After verifying your email, sign in to start using ProfeVision.

### Sign in

1. Go to the login page
2. Enter your email and password
3. Complete the security CAPTCHA
4. Click **Sign in**

![Login form](/images/getting-started/first-login-01-welcome-modal.webp)

### Welcome modal

On your first login you'll see a welcome modal with plan options:

- **Free Plan**: Basic features with limits
- **Plus Plan**: Unlimited features


You can choose:
- **Start for free**: Continue with the free plan
- **Upgrade to Plus**: Unlock all features

<Callout type="info">
You can upgrade to Plus at any time from settings.
</Callout>
