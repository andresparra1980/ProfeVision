## Générer le PDF

Générez des feuilles de réponses personnalisées pour chaque étudiant.

### Accès

1. Ouvrez l'examen que vous souhaitez imprimer
2. L'examen doit être **publié**
3. Cliquez sur **Télécharger le PDF**

![Download button](/images/printing/pdf-01-generate-button.webp)

### Sélectionner le groupe

1. Sélectionnez le groupe dans le menu déroulant
2. Le système génère un PDF avec une feuille par étudiant
3. Patientez pendant la génération (cela peut prendre quelques secondes)
4. Le PDF se télécharge automatiquement

![PDF preview](/images/printing/pdf-02-preview.webp)

### Contenu du PDF

Chaque feuille de réponses comprend :

- Un **QR code unique** pour identifier l'étudiant
- Les **données de l'étudiant** : nom, identification, groupe
- Les **données de l'examen** : titre, matière, durée
- Les **bulles de réponses** : organisées en 2 colonnes
- Des **repères de coin** : pour le calibrage de la numérisation

### Pages multiples

Si l'examen comporte plus de 40 questions, plusieurs pages sont générées par étudiant avec un indicateur de page.

<Callout type="info">
Les PDF sont générés et triés par ordre alphabétique selon le nom de famille de l'étudiant.
</Callout>
