## Diretrizes de impressão

Siga estas recomendações para garantir uma digitalização bem-sucedida.

### Papel recomendado

| Característica | Recomendação |
|----------------|--------------|
| Tamanho | Carta ou A4 |
| Cor | Branco |
| Gramatura | 75-90 g/m² (papel comum) |

### Configurações de impressão

1. **Escala**: 100% (não ajustar ao papel)
2. **Orientação**: Retrato
3. **Margens**: Use as margens do documento
4. **Cor**: Preto e branco é suficiente

![Settings](/images/printing/print-01-settings.webp)

### Verificações antes de imprimir

- [ ] O QR code está legível na visualização
- [ ] Os círculos aparecem completos
- [ ] Os marcadores de canto estão visíveis
- [ ] Sem cortes nas bordas

### O que evitar

| Problema | Consequência |
|----------|--------------|
| Redimensionar o documento | QR code e círculos distorcidos |
| Imprimir em papel colorido | Detecção mais difícil |
| Baixa qualidade de impressão | Círculos borrados |
| Dobrar as folhas | Danifica o QR code e os marcadores |

### Para os alunos

Instrua seus alunos a:

- Usar **lápis escuro** ou caneta preta
- Preencher os círculos **completamente**
- Não fazer marcas fora dos círculos
- Não dobrar ou danificar a folha

<Callout type="warn">
A má qualidade de impressão pode causar erros na digitalização.
</Callout>
