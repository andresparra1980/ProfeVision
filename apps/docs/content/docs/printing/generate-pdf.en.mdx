## Generate PDF

Generate personalized answer sheets for each student.

### Access

1. Open the exam you want to print
2. The exam must be **published**
3. Click **Download PDF**

![Download button](/images/printing/pdf-01-generate-button.webp)

### Select group

1. Select the group from the dropdown menu
2. The system generates a PDF with one sheet per student
3. Wait while it generates (may take a few seconds)
4. The PDF downloads automatically

![PDF preview](/images/printing/pdf-02-preview.webp)

### PDF content

Each answer sheet includes:

- **Unique QR** to identify the student
- **Student data**: name, identification, group
- **Exam data**: title, subject, duration
- **Answer bubbles**: organized in 2 columns
- **Corner markers**: for scan calibration

### Multiple pages

If the exam has more than 40 questions, multiple pages are generated per student with a page indicator.

<Callout type="info">
PDFs are generated sorted alphabetically by student's last name.
</Callout>
