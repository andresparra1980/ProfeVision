## Generar PDF

Genera las hojas de respuesta personalizadas para cada estudiante.

### Acceder

1. Abre el examen que quieres imprimir
2. El examen debe estar **publicado**
3. Haz clic en **Descargar PDF**

![Botón descargar](/images/printing/pdf-01-generate-button.webp)

### Seleccionar grupo

1. Selecciona el grupo del menú desplegable
2. El sistema genera un PDF con una hoja por estudiante
3. Espera mientras se genera (puede tomar unos segundos)
4. El PDF se descarga automáticamente

![Vista previa PDF](/images/printing/pdf-02-preview.webp)

### Contenido del PDF

Cada hoja de respuesta incluye:

- **QR único** para identificar al estudiante
- **Datos del estudiante**: nombre, identificación, grupo
- **Datos del examen**: título, materia, duración
- **Burbujas de respuesta**: organizadas en 2 columnas
- **Marcadores de esquina**: para calibración del escaneo

### Múltiples páginas

Si el examen tiene más de 40 preguntas, se generan múltiples páginas por estudiante con indicador de página.

<Callout type="info">
Los PDFs se generan ordenados alfabéticamente por apellido del estudiante.
</Callout>
