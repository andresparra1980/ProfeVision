## Gerar PDF

Gere folhas de respostas personalizadas para cada aluno.

### Acesso

1. Abra o exame que você deseja imprimir
2. O exame deve estar **publicado**
3. Clique em **Download PDF**

![Download button](/images/printing/pdf-01-generate-button.webp)

### Selecionar turma

1. Selecione a turma no menu suspenso
2. O sistema gera um PDF com uma folha por aluno
3. Aguarde enquanto o arquivo é gerado (pode levar alguns segundos)
4. O download do PDF será iniciado automaticamente

![PDF preview](/images/printing/pdf-02-preview.webp)

### Conteúdo do PDF

Cada folha de respostas inclui:

- **QR Code exclusivo** para identificar o aluno
- **Dados do aluno**: nome, identificação, turma
- **Dados do exame**: título, disciplina, duração
- **Bolhas de resposta**: organizadas em 2 colunas
- **Marcadores de canto**: para calibração da digitalização

### Múltiplas páginas

Se o exame tiver mais de 40 questões, serão geradas múltiplas páginas por aluno com um indicador de página.

<Callout type="info">
Os PDFs são gerados em ordem alfabética pelo sobrenome do aluno.
</Callout>
