## Directives d'impression

Suivez ces recommandations pour garantir une numérisation réussie.

### Papier recommandé

| Caractéristique | Recommandation |
|----------------|----------------|
| Taille | Lettre ou A4 |
| Couleur | Blanc |
| Grammage | 75-90 g/m² (papier normal) |

### Paramètres d'impression

1. **Échelle** : 100 % (ne pas adapter au papier)
2. **Orientation** : Portrait
3. **Marges** : Utilisez les marges du document
4. **Couleur** : Le noir et blanc est suffisant

![Settings](/images/printing/print-01-settings.webp)

### Vérifications avant l'impression

- [ ] Le QR code est lisible dans l'aperçu
- [ ] Les bulles apparaissent complètes
- [ ] Les repères de coins sont visibles
- [ ] Aucune coupure sur les bords

### Ce qu'il faut éviter

| Problème | Conséquence |
|---------|-------------|
| Modifier l'échelle du document | QR code et bulles déformés |
| Imprimer sur du papier couleur | Détection plus difficile |
| Faible qualité d'impression | Bulles floues |
| Plier les feuilles | Endommage le QR code et les repères |

### Pour les étudiants

Dites à vos étudiants de :

- Utiliser un **crayon foncé** ou un stylo noir
- Remplir **complètement** les bulles
- Ne pas faire de marques en dehors des bulles
- Ne pas plier ni endommager la feuille

<Callout type="warn">
Une mauvaise qualité d'impression peut entraîner des erreurs de numérisation.
</Callout>
