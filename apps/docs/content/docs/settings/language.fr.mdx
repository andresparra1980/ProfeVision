## Langue

ProfeVision est disponible en espagnol et en anglais.

### Changer de langue

1. Dans la barre latérale, recherchez le sélecteur de langue
2. Cliquez sur **ES** ou **EN**
3. La page se recharge dans la nouvelle langue


### Langues disponibles

| Code | Langue |
|------|----------|
| ES | Español |
| EN | English |

### Persistance

Votre préférence linguistique est enregistrée et appliquée à toutes vos sessions.

### Documentation

Cette documentation est également disponible dans les deux langues. Utilisez le sélecteur situé dans le coin supérieur.

<Callout type="info">
La langue affecte l'ensemble de l'interface, y compris les messages, les boutons et les menus.
</Callout>
