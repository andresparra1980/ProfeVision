## Language

ProfeVision is available in Spanish and English.

### Change language

1. In the sidebar, find the language selector
2. Click **ES** or **EN**
3. Page reloads in the new language

![Language selector](/images/settings/language-01-selector.webp)

### Available languages

| Code | Language |
|------|----------|
| ES | Español |
| EN | English |

### Persistence

Your language preference is saved and applied across all sessions.

### Documentation

This documentation is also available in both languages. Use the selector in the top corner.

<Callout type="info">
Language affects the entire interface, including messages, buttons, and menus.
</Callout>
