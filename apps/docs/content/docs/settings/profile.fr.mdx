## Profil

Mettez à jour vos informations personnelles.

### Champs modifiables

| Champ | Description |
|-------|-------------|
| Prénom | Votre prénom |
| Nom | Votre nom de famille |
| E-mail | Votre adresse e-mail (lecture seule) |

### Modifier le profil

1. Allez dans **Paramètres**
2. Modifiez les champs
3. Cliquez sur **Enregistrer**


### E-mail

Votre adresse e-mail ne peut pas être modifiée directement. Si vous avez besoin de la changer, contactez le support.

<Callout type="info">
Votre nom apparaît sur les feuilles de réponses et les rapports.
</Callout>
