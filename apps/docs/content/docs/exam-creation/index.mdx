## Creación de exámenes

ProfeVision ofrece múltiples formas de crear exámenes según tus necesidades.

### Métodos de creación

| Método | Descripción | Mejor para |
|--------|-------------|------------|
| Manual | Formulario tradicional | Control total |
| IA por pregunta | Genera preguntas individualmente | Inspiración rápida |
| Importar documento | Extrae de PDF/Word | Material existente |
| Chat con IA | Conversación guiada | Creación desde cero |
