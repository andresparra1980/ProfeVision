## Criação de Exames

O ProfeVision oferece várias maneiras de criar exames com base nas suas necessidades.

### Métodos de criação

| Método | Descrição | Recomendado para |
|--------|-----------|---------------|
| Importar documento | Extrair de PDF/Word | Material existente |
| Chat de IA | Conversa guiada | Criação do zero |
| Exame similar | Gerar variantes | Exames existentes |
