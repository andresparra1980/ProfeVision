## Creación manual

El método tradicional para crear exámenes con control total.

### Acceder

1. En la barra lateral, haz clic en **Exámenes**
2. Haz clic en **Crear examen**

![Lista de exámenes](/images/exam-creation/manual-01-list.webp)

### Información general

Completa los datos básicos:

- **Título**: Nombre del examen (mínimo 3 caracteres)
- **Descripción**: Descripción opcional
- **Instrucciones**: Instrucciones para los estudiantes
- **Materia**: Selecciona la materia
- **Grupo**: Selecciona el grupo (filtrado por materia)
- **Duración**: Tiempo en minutos (1-240)
- **Puntaje total**: Valor máximo (1-100)
- **Número de preguntas**: Cantidad de preguntas (1-40)

![Información general](/images/exam-creation/manual-02-info.webp)

### Agregar preguntas

Para cada pregunta:

1. Escribe el texto de la pregunta
2. Agrega las opciones de respuesta (2-4)
3. Marca la respuesta correcta
4. Repite para cada pregunta

![Agregar pregunta](/images/exam-creation/manual-03-question.webp)

### Tipos de pregunta

| Tipo | Descripción |
|------|-------------|
| Opción múltiple | Una respuesta correcta |
| Selección múltiple | Varias respuestas correctas |
| Verdadero/Falso | Dos opciones |

### Guardar

Haz clic en **Crear examen** para guardar.

<Callout type="info">
Tu progreso se guarda automáticamente en el navegador. Si cierras la página, puedes continuar después.
</Callout>
