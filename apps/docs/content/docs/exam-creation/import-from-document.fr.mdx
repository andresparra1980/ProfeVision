## Importer un document

Convertissez automatiquement vos examens existants au format numérique.

### Formats pris en charge

- PDF
- Word (.doc, .docx)

### Accès

1. Dans la barre latérale, cliquez sur **Examens**
2. Cliquez sur **Importer**

![Import button](/images/exam-creation/import-01-button.webp)

### Télécharger le fichier

1. Faites glisser le fichier vers la zone indiquée
2. Ou cliquez pour sélectionner le fichier
3. Patientez pendant le traitement

![Upload file](/images/exam-creation/import-02-upload.webp)

### Processus d'extraction

Le système :

1. Lit le document
2. Identifie les questions
3. Extrait les options de réponse
4. Détecte les bonnes réponses (si elles sont marquées)

Vous verrez des indicateurs de progression durant le processus.

![Progress](/images/exam-creation/import-03-progress.webp)

### Réviser les résultats

1. Examinez l'aperçu des questions extraites
2. Cliquez sur **Importer** pour continuer
3. Vous serez redirigé vers le formulaire de création
4. Modifiez les questions si nécessaire
5. Complétez les informations de l'examen
6. Enregistrez l'examen

![Preview](/images/exam-creation/import-04-preview.webp)

<Callout type="warn">
Révisez toujours les questions importées. L'extraction automatique peut nécessiter des ajustements.
</Callout>
