## Exam Creation

ProfeVision offers multiple ways to create exams based on your needs.

### Creation methods

| Method | Description | Best for |
|--------|-------------|----------|
| Manual | Traditional form | Full control |
| AI per question | Generate questions individually | Quick inspiration |
| Import document | Extract from PDF/Word | Existing material |
| AI chat | Guided conversation | Creation from scratch |
