## Similar exam

Generate a new version of an existing exam using artificial intelligence.

### When to use

- You need multiple versions of an exam
- You want to prevent students from copying
- You want to practice with similar variations

### Access

1. In **Exams**, find the base exam
2. Click the **Create similar exam** button (wand icon)

![Similar exam button](/images/exam-creation/similar-01-button.webp)

### Generation process

With a single click, AI starts automatically.

AI executes 6 steps:

1. **Analyze**: Reads the original exam
2. **Generate**: Creates similar questions
3. **Validate**: Checks quality
4. **Apply**: Adjusts recommendations
5. **Randomize**: Shuffles question order
6. **Finalize**: Prepares the draft

You'll see real-time progress.

![Progress](/images/exam-creation/similar-02-progress.webp)

### Result

On completion:
- A new exam is created as **draft**
- It opens automatically for editing
- Review and adjust generated questions

<Callout type="info">
The similar exam maintains the structure and difficulty of the original but with different questions.
</Callout>
