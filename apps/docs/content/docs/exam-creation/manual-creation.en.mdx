## Manual creation

The traditional method to create exams with full control.

### Access

1. In the sidebar, click **Exams**
2. Click **Create exam**

![Exams list](/images/exam-creation/manual-01-list.webp)

### General information

Fill in the basic data:

- **Title**: Exam name (minimum 3 characters)
- **Description**: Optional description
- **Instructions**: Instructions for students
- **Subject**: Select the subject
- **Group**: Select the group (filtered by subject)
- **Duration**: Time in minutes (1-240)
- **Total score**: Maximum value (1-100)
- **Number of questions**: Question count (1-40)

![General information](/images/exam-creation/manual-02-info.webp)

### Add questions

For each question:

1. Write the question text
2. Add answer options (2-4)
3. Mark the correct answer
4. Repeat for each question

![Add question](/images/exam-creation/manual-03-question.webp)

### Question types

| Type | Description |
|------|-------------|
| Multiple choice | One correct answer |
| Multiple selection | Several correct answers |
| True/False | Two options |

### Save

Click **Create exam** to save.

<Callout type="info">
Your progress is automatically saved in the browser. If you close the page, you can continue later.
</Callout>
