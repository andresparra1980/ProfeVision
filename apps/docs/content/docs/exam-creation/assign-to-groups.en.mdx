## Assign to groups

Connect an exam with one or more groups to apply it.

### Access

1. On the exam page, click **Assign**
2. Or from the exams list, select **Assign to groups**

![Assign access](/images/exam-creation/assign-01-access.webp)

### Select groups

1. You'll see active groups from the same subject
2. Check the groups where you'll apply the exam
3. For each selected group, configure:
   - **Application date**: When it will be applied
   - **Duration**: Time in minutes (defaults to exam's duration)

![Select groups](/images/exam-creation/assign-02-select.webp)

### Configure per group

Each group can have different settings:

| Field | Description |
|-------|-------------|
| Date | Application day and time |
| Duration | Available minutes |

![Configuration](/images/exam-creation/assign-03-config.webp)

### Save

1. Click **Save**
2. Groups are linked to the exam
3. You can generate PDF with correct QR for each group

<Callout type="info">
Only active groups from the exam's subject are shown.
</Callout>
