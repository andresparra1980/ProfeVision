## Création d'examen

ProfeVision propose plusieurs méthodes pour créer des examens en fonction de vos besoins.

### Méthodes de création

| Méthode | Description | Idéal pour |
|--------|-------------|----------|
| Importer un document | Extraire depuis un PDF/Word | Matériel existant |
| Chat IA | Conversation guidée | Création de zéro |
| Examen similaire | Générer des variantes | Examens existants |
