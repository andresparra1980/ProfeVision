## Importar documento

Converta exames existentes para o formato digital automaticamente.

### Formatos suportados

- PDF
- Word (.doc, .docx)

### Acesso

1. Na barra lateral, clique em **Exames**
2. Clique em **Importar**

![Import button](/images/exam-creation/import-01-button.webp)

### Carregar arquivo

1. Arraste o arquivo para a área indicada
2. Ou clique para selecionar o arquivo
3. Aguarde enquanto o processamento é realizado

![Upload file](/images/exam-creation/import-02-upload.webp)

### Processo de extração

O sistema:

1. Lê o documento
2. Identifica as questões
3. Extrai as opções de resposta
4. Detecta as respostas corretas (se estiverem marcadas)

Você verá indicadores de progresso durante o processo.

![Progress](/images/exam-creation/import-03-progress.webp)

### Revisar resultados

1. Revise a pré-visualização das questões extraídas
2. Clique em **Importar** para continuar
3. Você será redirecionado para o formulário de criação
4. Edite as questões se necessário
5. Complete as informações do exame
6. Salve o exame

![Preview](/images/exam-creation/import-04-preview.webp)

<Callout type="warn">
Sempre revise as questões importadas. A extração automática pode exigir ajustes.
</Callout>
