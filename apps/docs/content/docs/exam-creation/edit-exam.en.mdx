## Edit exam

Modify your exams after creating them.

### Access

1. In **Exams**, find the exam
2. Click **Edit**

![Edit access](/images/exam-creation/edit-01-access.webp)

### Draft exams

While the exam is in draft you can:

- Edit question text
- Change answer options
- Modify correct answer
- Add or delete questions
- Reorder questions

![Edit draft](/images/exam-creation/edit-02-draft.webp)

### Publish exam

When the exam is ready:

1. Click **Publish**
2. Status changes from "Draft" to "Published"
3. You can generate PDF and assign to groups

<Callout type="warn">
After publishing, edit options are limited to protect results integrity.
</Callout>

### Published exams

In published exams you can:

- **Enable/disable questions**: Question is excluded from calculation
- **Change correct answer**: Automatically recalculates grades

![Published exam](/images/exam-creation/edit-03-published.webp)

### Change correct answer

1. Click on the option that should be correct
2. Confirm the change
3. System recalculates all grades

<Callout type="info">
When changing the correct answer, all student grades are automatically updated.
</Callout>
