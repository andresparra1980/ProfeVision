## Examen similaire

Générez une nouvelle version d'un examen existant en utilisant l'intelligence artificielle.

### Quand l'utiliser

- Vous avez besoin de plusieurs versions d'un examen
- Vous souhaitez empêcher les étudiants de copier
- Vous voulez proposer des exercices avec des variantes similaires

### Accès

1. Dans **Exams**, recherchez l'examen de base
2. Cliquez sur le bouton **Créer un examen similaire** (icône baguette magique)

![Similar exam button](/images/exam-creation/similar-01-button.webp)

### Processus de génération

En un seul clic, l'IA démarre automatiquement.

L'IA exécute 6 étapes :

1. **Analyser** : Lit l'examen original
2. **Générer** : Crée des questions similaires
3. **Valider** : Vérifie la qualité
4. **Appliquer** : Ajuste les recommandations
5. **Mélanger** : Aléatorise l'ordre des questions
6. **Finaliser** : Prépare le brouillon

Vous verrez la progression en temps réel.

![Progress](/images/exam-creation/similar-02-progress.webp)

### Résultat

Une fois terminé :
- Un nouvel examen est créé en tant que **brouillon**
- Il s'ouvre automatiquement pour édition
- Révisez et ajustez les questions générées

<Callout type="info">
L'examen similaire conserve la structure et la difficulté de l'original, mais avec des questions différentes.
</Callout>
