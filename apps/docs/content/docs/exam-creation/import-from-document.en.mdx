## Import document

Convert existing exams to digital format automatically.

### Supported formats

- PDF
- Word (.doc, .docx)

### Access

1. In the sidebar, click **Exams**
2. Click **Import**

![Import button](/images/exam-creation/import-01-button.webp)

### Upload file

1. Drag the file to the indicated area
2. Or click to select the file
3. Wait while it processes

![Upload file](/images/exam-creation/import-02-upload.webp)

### Extraction process

The system:

1. Reads the document
2. Identifies questions
3. Extracts answer options
4. Detects correct answers (if marked)

You'll see progress indicators during the process.

![Progress](/images/exam-creation/import-03-progress.webp)

### Review results

1. Review the preview of extracted questions
2. Click **Import** to continue
3. You'll be redirected to the creation form
4. Edit questions if needed
5. Complete the exam information
6. Save the exam

![Preview](/images/exam-creation/import-04-preview.webp)

<Callout type="warn">
Always review imported questions. Automatic extraction may require adjustments.
</Callout>
