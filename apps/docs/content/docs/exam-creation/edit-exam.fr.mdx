## Modifier l'examen

Modifiez vos examens après les avoir créés.

### Accès

1. Dans **Examens**, recherchez l'examen
2. Cliquez sur **Modifier**

![Edit access](/images/exam-creation/edit-01-access.webp)

### Examens en brouillon

Tant que l'examen est en mode brouillon, vous pouvez :

- Modifier le texte des questions
- Changer les options de réponse
- Modifier la réponse correcte
- Ajouter ou supprimer des questions
- Réorganiser les questions

![Edit draft](/images/exam-creation/edit-02-draft.webp)

### Publier l'examen

Lorsque l'examen est prêt :

1. Cliquez sur **Publier**
2. Le statut passe de "Brouillon" à "Publié"
3. Vous pouvez générer le PDF et l'assigner à des groupes

<Callout type="warn">
Après la publication, les options de modification sont limitées pour protéger l'intégrité des résultats.
</Callout>

### Examens publiés

Dans les examens publiés, vous pouvez :

- **Activer/désactiver des questions** : La question est exclue du calcul
- **Changer la réponse correcte** : Recalcule automatiquement la notation

![Published exam](/images/exam-creation/edit-03-published.webp)

### Changer la réponse correcte

1. Cliquez sur l'option qui devrait être correcte
2. Confirmez le changement
3. Le système recalcule toutes les notations

<Callout type="info">
Lors du changement de la réponse correcte, toutes les notations des étudiants sont automatiquement mises à jour.
</Callout>
