## Importar documento

Convierte exámenes existentes en formato digital automáticamente.

### Formatos soportados

- PDF
- Word (.doc, .docx)

### Acceder

1. En la barra lateral, haz clic en **Exámenes**
2. Haz clic en **Importar**

![Botón importar](/images/exam-creation/import-01-button.webp)

### Subir archivo

1. Arrastra el archivo al área indicada
2. O haz clic para seleccionar el archivo
3. Espera mientras se procesa

![Subir archivo](/images/exam-creation/import-02-upload.webp)

### Proceso de extracción

El sistema:

1. Lee el documento
2. Identifica las preguntas
3. Extrae las opciones de respuesta
4. Detecta las respuestas correctas (si están marcadas)

Verás indicadores de progreso durante el proceso.

![Progreso](/images/exam-creation/import-03-progress.webp)

### Revisar resultados

1. Revisa la vista previa de preguntas extraídas
2. Haz clic en **Importar** para continuar
3. Serás redirigido al formulario de creación
4. Edita las preguntas si es necesario
5. Completa la información del examen
6. Guarda el examen

![Vista previa](/images/exam-creation/import-04-preview.webp)

<Callout type="warn">
Revisa siempre las preguntas importadas. La extracción automática puede requerir ajustes.
</Callout>
