## Affecter à des groupes

Associez un examen à un ou plusieurs groupes pour l'appliquer.

### Accès

1. Sur la page de l'examen, cliquez sur **Assign**
2. Ou depuis la liste des examens, sélectionnez **Assign to groups**

![Assign access](/images/exam-creation/assign-01-access.webp)

### Sélectionner les groupes

1. Vous verrez les groupes actifs de la même matière
2. Cochez les groupes où vous appliquerez l'examen
3. Pour chaque groupe sélectionné, configurez :
   - **Application date** : Quand il sera appliqué
   - **Duration** : Temps en minutes (par défaut la durée de l'examen)

![Select groups](/images/exam-creation/assign-02-select.webp)

### Configurer par groupe

Chaque groupe peut avoir des paramètres différents :

| Champ | Description |
|-------|-------------|
| Date | Jour et heure d'application |
| Duration | Minutes disponibles |

![Configuration](/images/exam-creation/assign-03-config.webp)

### Enregistrer

1. Cliquez sur **Save**
2. Les groupes sont liés à l'examen
3. Vous pouvez générer le PDF avec le QR correct pour chaque groupe

<Callout type="info">
Seuls les groupes actifs de la matière de l'examen sont affichés.
</Callout>
