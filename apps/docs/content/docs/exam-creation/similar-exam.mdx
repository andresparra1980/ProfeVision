## Examen similar

Genera una nueva versión de un examen existente usando inteligencia artificial.

### Cuándo usar

- Necesitas múltiples versiones de un examen
- Quieres evitar que estudiantes copien
- Deseas practicar con variaciones similares

### Acceder

1. En **Exámenes**, encuentra el examen base
2. Haz clic en el botón **Crear examen similar** (ícono de varita)

![Botón examen similar](/images/exam-creation/similar-01-button.webp)

### Proceso de generación

Con un solo clic, la IA comienza automáticamente.

La IA ejecuta 6 pasos:

1. **Analizar**: Lee el examen original
2. **Generar**: Crea preguntas similares
3. **Validar**: Verifica calidad
4. **Aplicar**: Ajusta recomendaciones
5. **Aleatorizar**: Mezcla orden de preguntas
6. **Finalizar**: Prepara el borrador

Verás el progreso en tiempo real.

![Progreso](/images/exam-creation/similar-02-progress.webp)

### Resultado

Al completar:
- Se crea un nuevo examen en **borrador**
- Se abre automáticamente para edición
- Revisa y ajusta las preguntas generadas

<Callout type="info">
El examen similar mantiene la estructura y dificultad del original pero con preguntas diferentes.
</Callout>
