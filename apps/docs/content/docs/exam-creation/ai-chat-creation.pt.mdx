## Chat de IA

Crie exames de forma conversacional com assistência de IA.

### Acesso

Para abrir o Chat de IA, siga estas etapas:

1. Faça login na sua conta no ProfeVision.
2. Navegue até o **painel** principal.
3. No menu lateral, clique em **Chat de IA**.

## Principais Funcionalidades

O Chat de IA oferece diversas capacidades para otimizar sua rotina:

- **Geração de Questões:** Peça à IA para criar questões de múltipla escolha ou dissertativas com base em um tópico específico.
- **Suporte na Correção:** Obtenha sugestões sobre como **corrigir** respostas abertas de forma justa e consistente.
- **Estruturação de Exames:** Organize a ordem e a dificuldade das questões para cada **aluno**.
- **Assistência com Digitalização:** Tire dúvidas sobre como **digitalizar** as folhas de respostas corretamente para que o sistema processe as notas.

## Dicas para melhores resultados

Para obter as melhores respostas do Chat de IA, lembre-se:

- **Seja específico:** Informe a série ou o nível de conhecimento do **aluno**.
- **Defina o formato:** Indique se você prefere um **exame** curto ou uma avaliação abrangente.
- **Contextualize:** Forneça o conteúdo programático para que a IA gere questões alinhadas com o que foi ensinado.

```javascript
// Exemplo de como a integração do chat funciona internamente
const aiAssistant = new ProfeVisionAI({
  role: "teacher-assistant",
  language: "pt-BR"
});
```

Se você encontrar qualquer problema ao usar o Chat de IA, consulte nossa página de [suporte](https://profevision.com/support) ou entre em contato com a equipe técnica.

## Chat de IA

Crie exames de forma conversacional com a assistência da IA.

### Acesso

1. Na barra lateral, clique em **Exames**
2. Clique em **Criar com IA**

![AI chat access](/images/exam-creation/chat-01-access.webp)

### Como funciona

1. Descreva o exame que você precisa
2. A IA faz perguntas para entender melhor
3. Gera as questões com base na sua conversa
4. Revise e ajuste o resultado

![Chat interface](/images/exam-creation/chat-02-interface.webp)

### Exemplo de conversa

> **Você**: Preciso de um exame de matemática sobre frações para o 5º ano
>
> **IA**: Entendido. De quantas questões você precisa? Quais operações você deseja incluir?
>
> **Você**: 10 questões, adição e subtração de frações
>
> **IA**: Perfeito. Gerando o exame...

### Opções disponíveis

- **Idioma**: Altere o idioma da conversa
- **Limpar chat**: Exclua a conversa e recomece
- **Salvar rascunho**: Salve o progresso para continuar mais tarde

![Options](/images/exam-creation/chat-03-options.webp)

### Salvar rascunho

1. Clique em **Salvar rascunho**
2. Escolha um nome para o rascunho
3. Continue mais tarde de onde você parou

<Callout type="info">
Os rascunhos são salvos em sua conta e você pode acessá-los a partir da lista de exames.
</Callout>

## Abordagem Pedagógica

Esta ferramenta não improvisa; ela simula o processo cognitivo de um designer instrucional especialista.

### Princípios Chave

- **Design Reverso (Backward Design)**: O sistema define primeiro os objetivos de aprendizagem (Planejamento) e, em seguida, constrói instrumentos alinhados a esses objetivos. Isso garante a **validade do conteúdo** e evita avaliações enviesadas.
- **Taxonomia de Bloom**: Cada questão visa um nível cognitivo específico (Lembrar, Entender, Aplicar, Analisar, Avaliar), permitindo avaliar **competências** reais em vez de apenas memorização superficial.
- **Zona de Desenvolvimento Proximal (Vygotsky)**: Através de configurações de dificuldade "mistas", criamos andaimes (scaffolding) que guiam o aluno desde conceitos básicos até análises complexas.
- **Avaliação Formativa**: A inclusão de uma "justificativa" (rationale) para cada resposta transforma o exame em uma ferramenta de aprendizagem, fornecendo feedback imediato que reforça a metacognição.
