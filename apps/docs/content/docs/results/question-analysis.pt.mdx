## Análise de questões

Identifique quais questões foram mais fáceis ou difíceis para seus alunos.

### Gráfico de taxa de sucesso

O gráfico de barras mostra a porcentagem de respostas corretas por questão.

**Cores**:
- **Verde** (≥80%): Questão fácil
- **Amarelo** (50-79%): Dificuldade média
- **Vermelho** (<50%): Questão difícil

![Gráfico de sucesso](/images/results/results-02-question-chart.webp)

### Estatísticas

| Métrica | Significado |
|---------|-------------|
| Sucesso médio | % média de respostas corretas |
| Questão mais fácil | Maior porcentagem de acerto |
| Questão mais difícil | Menor porcentagem de acerto |
| Total analisado | Questões com pelo menos uma resposta |

### Distribuição de respostas

O gráfico de barras empilhadas mostra qual opção cada aluno escolheu.

![Distribuição](/images/results/results-03-answer-distribution.webp)

**Informações exibidas**:
- Contagem para cada opção (A, B, C, D...)
- Porcentagem por opção
- Resposta correta sinalizada
- Alunos sem resposta

<Callout type="info">
Utilize esta análise para identificar conceitos que precisam de reforço em sala de aula.
</Callout>
