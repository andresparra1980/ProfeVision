## Manual grading

For special cases where scanning isn't possible.

### When to use

- Student lost their answer sheet
- Technical problems with scanning
- Oral or special evaluations
- Grade adjustments

### Enter grade

1. In the students table
2. Find the ungraded student
3. Click **Enter grade**
4. Enter the score (0-5)
5. Confirm

![Manual grading](/images/results/results-06-manual-edit.webp)

### Scale

| Value | Description |
|-------|-------------|
| 0.0 | Minimum |
| 5.0 | Maximum |
| 0.1 | Minimum increment |

### Warning

Manual grades:
- Have no backup image
- Don't show individual answers
- Are marked as "Manual grade"

<Callout type="warn">
Use manual grading only when scanning isn't possible.
</Callout>
