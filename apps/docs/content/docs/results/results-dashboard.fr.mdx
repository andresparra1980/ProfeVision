## Tableau de bord des résultats

Le tableau de bord présente un résumé complet des résultats de l'examen.

### Accès

1. Dans **Exams**, sélectionnez un examen
2. Cliquez sur **Voir les résultats**

### Informations sur l'examen

En haut, vous trouverez :

- Le titre de l'examen
- La matière
- Le score total
- La date de création

### Statistiques générales

| Métrique | Description |
|----------|-------------|
| Étudiants notés | Nombre d'étudiants ayant passé l'examen |
| Moyenne | Note moyenne du groupe |
| Note la plus haute | Meilleure note obtenue |
| Note la plus basse | Note la plus basse obtenue |

![Results dashboard](/images/results/results-01-dashboard.webp)

### Sélectionner un groupe

Si l'examen est assigné à plusieurs groupes :

1. Utilisez le sélecteur de groupe
2. Les données se mettent à jour automatiquement
3. Votre sélection est mémorisée

<Callout type="info">
Le système mémorise le dernier groupe sélectionné pour chaque examen.
</Callout>
