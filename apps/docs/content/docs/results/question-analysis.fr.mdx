## Analyse des questions

Identifiez quelles questions ont été les plus faciles ou les plus difficiles pour vos étudiants.

### Graphique du taux de réussite

Le diagramme à barres affiche le pourcentage de bonnes réponses par question.

**Couleurs** :
- **Vert** (≥80%) : Question facile
- **Jaune** (50-79%) : Difficulté moyenne
- **Rouge** (&lt;50%) : Question difficile

![Success chart](/images/results/results-02-question-chart.webp)

### Statistiques

| Métrique | Signification |
|--------|---------|
| Succès moyen | % moyen de réponses correctes |
| Question la plus facile | Pourcentage de réussite le plus élevé |
| Question la plus difficile | Pourcentage de réussite le plus bas |
| Total analysé | Questions avec au moins une réponse |

### Distribution des réponses

Le diagramme à barres empilées montre quelle option chaque étudiant a choisie.

![Distribution](/images/results/results-03-answer-distribution.webp)

**Informations affichées** :
- Nombre pour chaque option (A, B, C, D...)
- Pourcentage par option
- Réponse correcte indiquée
- Étudiants sans réponse

<Callout type="info">
Utilisez cette analyse pour identifier les concepts qui nécessitent un renforcement en classe.
</Callout>
