## Notation manuelle

Pour les cas particuliers où la numérisation n'est pas possible.

### Quand l'utiliser

- L'étudiant a perdu sa feuille d'examen
- Problèmes techniques lors de la numérisation
- Évaluations orales ou spéciales
- Ajustements de notes

### Saisir une note

1. Dans le tableau des étudiants
2. Trouvez l'étudiant non noté
3. Cliquez sur **Saisir une note**
4. Saisissez la note (0-5)
5. Confirmez

![Manual grading](/images/results/results-06-manual-edit.webp)

### Échelle

| Valeur | Description |
|-------|-------------|
| 0.0 | Minimum |
| 5.0 | Maximum |
| 0.1 | Incrément minimum |

### Avertissement

Les notations manuelles :
- N'ont pas d'image de sauvegarde
- Ne montrent pas les réponses individuelles
- Sont marquées comme « Notation manuelle »

<Callout type="warn">
Utilisez la notation manuelle uniquement lorsque la numérisation n'est pas possible.
</Callout>
