## Dashboard de resultados

El dashboard muestra un resumen completo de los resultados del examen.

### Acceder

1. En **Exámenes**, selecciona un examen
2. Haz clic en **Ver resultados**

### Información del examen

En la parte superior verás:

- Título del examen
- Materia
- Puntaje total
- Fecha de creación

### Estadísticas generales

| Métrica | Descripción |
|---------|-------------|
| Estudiantes calificados | Cuántos presentaron el examen |
| Promedio | Calificación promedio del grupo |
| Nota más alta | Mejor calificación obtenida |
| Nota más baja | Menor calificación obtenida |

![Dashboard de resultados](/images/results/results-01-dashboard.webp)

### Seleccionar grupo

Si el examen está asignado a múltiples grupos:

1. Usa el selector de grupo
2. Los datos se actualizan automáticamente
3. Tu selección se recuerda

<Callout type="info">
El sistema recuerda el último grupo seleccionado para cada examen.
</Callout>
