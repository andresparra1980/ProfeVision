## Student results

View each student's grade and their answers.

### Results table

The table shows:

| Column | Content |
|--------|---------|
| Name | Student's full name |
| ID | Identification number |
| Grade | Score obtained |
| Percentage | % correct answers |
| Status | Graded or Not submitted |
| Actions | View details, manual grade |

![Results table](/images/results/results-04-student-table.webp)

### Search student

Use the search field to filter by name.

### View details

1. Click **View details**
2. A dialog opens with 3 tabs:
   - **Answers**: Marked bubbles vs correct ones
   - **Original image**: Scanned photo
   - **Processed image**: Image with detection

### Correct answer

If you detect a detection error:

1. In the answers tab
2. Click on the correct bubble
3. Confirm the change
4. Grade recalculates

<Callout type="warn">
Correcting answers affects the student's grade immediately.
</Callout>
