## Résultats des étudiants

Consultez la note de chaque étudiant et ses réponses.

### Tableau des résultats

Le tableau affiche :

| Colonne | Contenu |
|--------|---------|
| Nom | Nom complet de l'étudiant |
| Identifiant | Numéro d'identification |
| Note | Score obtenu |
| Pourcentage | % de réponses correctes |
| Statut | Noté ou Non soumis |
| Actions | Voir les détails, notation manuelle |

![Results table](/images/results/results-04-student-table.webp)

### Rechercher un étudiant

Utilisez le champ de recherche pour filtrer par nom.

### Voir les détails

1. Cliquez sur **Voir les détails**
2. Une boîte de dialogue s'ouvre avec 3 onglets :
   - **Réponses** : Bulles cochées vs réponses correctes
   - **Image originale** : Photo numérisée
   - **Image traitée** : Image avec détection

### Corriger une réponse

Si vous détectez une erreur de détection :

1. Dans l'onglet des réponses
2. Cliquez sur la bulle correcte
3. Confirmez la modification
4. La note est recalculée

<Callout type="warn">
La correction des réponses affecte immédiatement la note de l'étudiant.
</Callout>
