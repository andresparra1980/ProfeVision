## Painel de resultados

O painel mostra um resumo completo dos resultados do exame.

### Acesso

1. Em **Exames**, selecione um exame
2. Clique em **Visualizar resultados**

### Informações do exame

No topo, você verá:

- Título do exame
- Disciplina
- Pontuação total
- Data de criação

### Estatísticas gerais

| Métrica | Descrição |
|--------|-------------|
| Alunos avaliados | Quantos realizaram o exame |
| Média | Média de notas do grupo |
| Maior nota | Melhor nota obtida |
| Menor nota | Menor nota obtida |

![Results dashboard](/images/results/results-01-dashboard.webp)

### Selecionar grupo

Se o exame estiver atribuído a vários grupos:

1. Use o seletor de grupo
2. Os dados são atualizados automaticamente
3. Sua seleção será lembrada

<Callout type="info">
O sistema lembra o último grupo selecionado para cada exame.
</Callout>
