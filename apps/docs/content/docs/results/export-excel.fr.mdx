## Export vers Excel

Téléchargez tous les résultats pour les utiliser dans d'autres outils.

### Exportation

1. Dans le tableau de bord des résultats
2. Cliquez sur **Exporter vers Excel**
3. Le fichier se télécharge automatiquement

![Export button](/images/results/results-05-export-button.webp)

### Contenu du fichier

Le fichier Excel comprend :

**En-tête** :
- Titre de l'examen
- Matière
- Groupe
- Date d'exportation

**Statistiques** :
- Moyenne du groupe
- Note la plus élevée
- Note la plus basse

**Tableau des étudiants** :
| Colonne | Contenu |
|---------|---------|
| Nom | Nom complet |
| Identifiant | Numéro d'identification |
| Note | Score obtenu |
| Pourcentage | % de réponses correctes |
| Date | Date de notation |

### Étudiants non notés

Les étudiants qui n'ont pas passé l'examen apparaissent avec :
- Note : « Non soumis »
- Pourcentage : vide
- Date : vide

<Callout type="info">
Les résultats sont triés par ordre alphabétique par nom de famille.
</Callout>
