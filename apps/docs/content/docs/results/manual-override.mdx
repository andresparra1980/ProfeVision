## Calificación manual

Para casos especiales donde no puedes usar el escaneo.

### Cuándo usar

- Estudiante perdió su hoja de respuesta
- Problemas técnicos con el escaneo
- Evaluaciones orales o especiales
- Ajustes de calificación

### Ingresar calificación

1. En la tabla de estudiantes
2. Busca al estudiante sin calificar
3. Haz clic en **Ingresar calificación**
4. Ingresa el puntaje (0-5)
5. Confirma

![Calificación manual](/images/results/results-06-manual-edit.webp)

### Escala

| Valor | Descripción |
|-------|-------------|
| 0.0 | Mínimo |
| 5.0 | Máximo |
| 0.1 | Incremento mínimo |

### Advertencia

Las calificaciones manuales:
- No tienen imagen de respaldo
- No muestran respuestas individuales
- Se marcan como "Calificación manual"

<Callout type="warn">
Usa la calificación manual solo cuando el escaneo no sea posible.
</Callout>
