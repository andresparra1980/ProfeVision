## Exportar para Excel

Baixe todos os resultados para utilizar em outras ferramentas.

### Exportar

1. No painel de resultados
2. Clique em **Exportar para Excel**
3. O arquivo será baixado automaticamente

![Export button](/images/results/results-05-export-button.webp)

### Conteúdo do arquivo

O arquivo Excel inclui:

**Cabeçalho**:
- Título do exame
- Disciplina
- Turma
- Data de exportação

**Estatísticas**:
- Média da turma
- Maior nota
- Menor nota

**Tabela de alunos**:
| Coluna | Conteúdo |
|--------|---------|
| Nome | Nome completo |
| ID | Número de identificação |
| Nota | Pontuação obtida |
| Porcentagem | % de acertos |
| Data | Data da correção |

### Alunos não avaliados

Os alunos que não realizaram o exame aparecem com:
- Nota: "Não entregue"
- Porcentagem: vazio
- Data: vazio

<Callout type="info">
Os resultados são organizados em ordem alfabética pelo sobrenome.
</Callout>
