## Question analysis

Identify which questions were easier or harder for your students.

### Success rate chart

The bar chart shows the percentage of correct answers per question.

**Colors**:
- **Green** (≥80%): Easy question
- **Yellow** (50-79%): Medium difficulty
- **Red** (&lt;50%): Difficult question

![Success chart](/images/results/results-02-question-chart.webp)

### Statistics

| Metric | Meaning |
|--------|---------|
| Average success | % average correct answers |
| Easiest question | Highest correct percentage |
| Hardest question | Lowest correct percentage |
| Total analyzed | Questions with at least one answer |

### Answer distribution

The stacked bar chart shows which option each student chose.

![Distribution](/images/results/results-03-answer-distribution.webp)

**Information shown**:
- Count for each option (A, B, C, D...)
- Percentage per option
- Correct answer marked
- Students without answer

<Callout type="info">
Use this analysis to identify concepts that need reinforcement in class.
</Callout>
