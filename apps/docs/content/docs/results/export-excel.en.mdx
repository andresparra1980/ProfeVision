## Export to Excel

Download all results to use in other tools.

### Export

1. In the results dashboard
2. Click **Export to Excel**
3. File downloads automatically

![Export button](/images/results/export-01-button.webp)

### File content

The Excel file includes:

**Header**:
- Exam title
- Subject
- Group
- Export date

**Statistics**:
- Group average
- Highest grade
- Lowest grade

**Student table**:
| Column | Content |
|--------|---------|
| Name | Full name |
| ID | Identification number |
| Grade | Score obtained |
| Percentage | % correct |
| Date | Grading date |

![Excel file](/images/results/export-02-file.webp)

### Ungraded students

Students who didn't take the exam appear with:
- Grade: "Not submitted"
- Percentage: empty
- Date: empty

<Callout type="info">
Results are sorted alphabetically by last name.
</Callout>
