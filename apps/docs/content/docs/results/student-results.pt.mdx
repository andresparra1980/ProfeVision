## Resultados dos alunos

Visualize a nota de cada aluno e suas respostas.

### Tabela de resultados

A tabela mostra:

| Coluna | Conteúdo |
|--------|---------|
| Nome | Nome completo do aluno |
| ID | Número de identificação |
| Nota | Pontuação obtida |
| Porcentagem | % de respostas corretas |
| Status | Corrigido ou Não enviado |
| Ações | Ver detalhes, nota manual |

![Results table](/images/results/results-04-student-table.webp)

### Buscar aluno

Use o campo de busca para filtrar por nome.

### Ver detalhes

1. Clique em **Ver detalhes**
2. Uma caixa de diálogo será aberta com 3 abas:
   - **Respostas**: Bolhas marcadas vs. corretas
   - **Imagem original**: Foto digitalizada
   - **Imagem processada**: Imagem com a detecção

### Corrigir resposta

Se você detectar um erro de detecção:

1. Na aba de respostas
2. Clique na bolha correta
3. Confirme a alteração
4. A nota será recalculada

<Callout type="warn">
A correção de respostas afeta a nota do aluno imediatamente.
</Callout>
