## Exportar a Excel

Descarga todos los resultados para usar en otras herramientas.

### Exportar

1. En el dashboard de resultados
2. Haz clic en **Exportar a Excel**
3. El archivo se descarga automáticamente

![Botón exportar](/images/results/results-05-export-button.webp)

### Contenido del archivo

El archivo Excel incluye:

**Encabezado**:
- Título del examen
- Materia
- Grupo
- Fecha de exportación

**Estadísticas**:
- Promedio del grupo
- Nota más alta
- Nota más baja

**Tabla de estudiantes**:
| Columna | Contenido |
|---------|-----------|
| Nombre | Nombre completo |
| Identificación | Número de ID |
| Calificación | Puntaje obtenido |
| Porcentaje | % de aciertos |
| Fecha | Fecha de calificación |

### Estudiantes sin calificar

Los estudiantes que no presentaron aparecen con:
- Calificación: "No presentó"
- Porcentaje: vacío
- Fecha: vacío

<Callout type="info">
Los resultados se ordenan alfabéticamente por apellido.
</Callout>
