## Resultados e Análises

Após a correção, analise os resultados para aprimorar o seu ensino.

### Funcionalidades

| Funcionalidade | Descrição |
|---------|-------------|
| Painel | Visão geral das estatísticas |
| Análise de questões | Identifique questões difíceis |
| Resultados individuais | Nota por aluno |
| Exportação | Download em Excel ou PDF |
| Correção manual | Para casos especiais |
| Sincronização | Conecte ao seu livro de notas |
