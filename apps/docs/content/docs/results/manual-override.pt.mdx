## Correção manual

Para casos especiais onde a digitalização não é possível.

### Quando utilizar

- O aluno perdeu sua folha de respostas
- Problemas técnicos com a digitalização
- Avaliações orais ou especiais
- Ajustes de nota

### Inserir nota

1. Na tabela de alunos
2. Encontre o aluno sem nota
3. Clique em **Inserir nota**
4. Insira a pontuação (0-5)
5. Confirme

![Manual grading](/images/results/results-06-manual-edit.webp)

### Escala

| Valor | Descrição |
|-------|-------------|
| 0.0 | Mínimo |
| 5.0 | Máximo |
| 0.1 | Incremento mínimo |

### Aviso

Notas manuais:
- Não possuem imagem de backup
- Não exibem respostas individuais
- São marcadas como "Nota manual"

<Callout type="warn">
Utilize a correção manual apenas quando a digitalização não for possível.
</Callout>
