## Análisis por pregunta

Identifica qué preguntas fueron más fáciles o difíciles para tus estudiantes.

### Gráfico de aciertos

El gráfico de barras muestra el porcentaje de aciertos por pregunta.

**Colores**:
- **Verde** (≥80%): Pregunta fácil
- **Amarillo** (50-79%): Dificultad media
- **Rojo** (&lt;50%): Pregunta difícil

![Gráfico de aciertos](/images/results/results-02-question-chart.webp)

### Estadísticas

| Métrica | Significado |
|---------|-------------|
| Promedio de aciertos | % promedio de respuestas correctas |
| Pregunta más fácil | Mayor porcentaje de aciertos |
| Pregunta más difícil | Menor porcentaje de aciertos |
| Total analizadas | Preguntas con al menos una respuesta |

### Distribución de respuestas

El gráfico de barras apiladas muestra qué opción eligió cada estudiante.

![Distribución](/images/results/results-03-answer-distribution.webp)

**Información mostrada**:
- Cantidad por cada opción (A, B, C, D...)
- Porcentaje de cada opción
- Respuesta correcta marcada
- Estudiantes sin respuesta

<Callout type="info">
Usa este análisis para identificar conceptos que necesitan refuerzo en clase.
</Callout>
