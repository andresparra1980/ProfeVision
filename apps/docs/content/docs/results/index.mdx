## Resultados y analíticas

Después de calificar, analiza los resultados para mejorar tu enseñanza.

### Funcionalidades

| Función | Descripción |
|---------|-------------|
| Dashboard | Vista general de estadísticas |
| Análisis por pregunta | Identifica preguntas difíciles |
| Resultados individuales | Calificación por estudiante |
| Exportar | Descarga en Excel o PDF |
| Calificación manual | Para casos especiales |
| Sincronizar | Conecta con el libro de calificaciones |
