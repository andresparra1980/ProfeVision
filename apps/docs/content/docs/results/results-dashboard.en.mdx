## Results dashboard

The dashboard shows a complete summary of exam results.

### Access

1. In **Exams**, select an exam
2. Click **View results**

### Exam information

At the top you'll see:

- Exam title
- Subject
- Total score
- Creation date

### General statistics

| Metric | Description |
|--------|-------------|
| Graded students | How many took the exam |
| Average | Group average grade |
| Highest grade | Best grade obtained |
| Lowest grade | Lowest grade obtained |

![Results dashboard](/images/results/results-01-dashboard.webp)

### Select group

If the exam is assigned to multiple groups:

1. Use the group selector
2. Data updates automatically
3. Your selection is remembered

<Callout type="info">
The system remembers the last selected group for each exam.
</Callout>
