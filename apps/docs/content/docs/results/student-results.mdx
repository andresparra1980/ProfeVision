## Resultados de estudiantes

Ve la calificación de cada estudiante y sus respuestas.

### Tabla de resultados

La tabla muestra:

| Columna | Contenido |
|---------|-----------|
| Nombre | Nombre completo del estudiante |
| Identificación | Número de identificación |
| Calificación | Puntaje obtenido |
| Porcentaje | % de respuestas correctas |
| Estado | Calificado o No presentó |
| Acciones | Ver detalles, calificar manual |

![Tabla de resultados](/images/results/results-04-student-table.webp)

### Buscar estudiante

Usa el campo de búsqueda para filtrar por nombre.

### Ver detalles

1. Haz clic en **Ver detalles**
2. Se abre un diálogo con 3 pestañas:
   - **Respuestas**: Burbujas marcadas vs correctas
   - **Imagen original**: Foto escaneada
   - **Imagen procesada**: Imagen con detección

### Corregir respuesta

Si detectas un error en la detección:

1. En la pestaña de respuestas
2. Haz clic en la burbuja correcta
3. Confirma el cambio
4. La calificación se recalcula

<Callout type="warn">
Corregir respuestas afecta la calificación del estudiante inmediatamente.
</Callout>
