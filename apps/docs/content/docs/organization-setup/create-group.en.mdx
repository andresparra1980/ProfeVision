## Create group

A group represents a class or section of students.

### Access groups

1. In the sidebar, click **Groups**
2. You'll see your groups displayed as cards

![Groups list](/images/organization-setup/group-01-list.webp)

### Add new group

1. Click the **New Group** button
2. Fill in the form:
   - **Name**: Group name (e.g., "3rd A")
   - **Institution**: Select an institution
   - **Subject**: Select a subject (filtered by institution)
   - **School period**: Start and end dates (optional)
   - **Description**: Optional description
3. Click **Create**

![Group form](/images/organization-setup/group-02-create-form.webp)

### Group actions

Each card has quick actions:
- **Edit**: Modify group data
- **Students**: Manage group students
- **Grades**: View the grades table
- **Scheme**: Configure the grading scale
- **Archive**: Hide the group without deleting


### Archived groups

Use the **Show archived** button to view groups from previous periods. You can reactivate them at any time.

<Callout type="info">
Archiving is better than deleting: you keep the data and can restore the group later.
</Callout>
