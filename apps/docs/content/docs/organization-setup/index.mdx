## Configuración organizacional

Antes de crear exámenes, necesitas configurar tu estructura organizacional.

### Jerarquía

ProfeVision usa una jerarquía de 3 niveles:

1. **Institución** → Tu escuela o universidad
2. **Materia** → Los cursos que enseñas
3. **Grupo** → Las clases de estudiantes
