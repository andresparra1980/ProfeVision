## Configuração da Organização

Antes de criar exames, você precisa configurar sua estrutura organizacional.

### Hierarquia

O ProfeVision utiliza uma hierarquia de 3 níveis:

1. **Instituição** → Sua escola ou universidade
2. **Disciplina** → Os cursos que você ministra
3. **Turma** → Seus grupos de alunos
