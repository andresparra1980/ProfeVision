## Criar grupo

Um grupo representa uma turma ou seção de alunos.

### Acessar grupos

1. Na barra lateral, clique em **Grupos**
2. Você verá seus grupos exibidos como cards

![Lista de grupos](/images/organization-setup/group-01-list.webp)

### Adicionar novo grupo

1. Clique no botão **Novo Grupo**
2. Preencha o formulário:
   - **Nome**: Nome do grupo (ex: "3º A")
   - **Instituição**: Selecione uma instituição
   - **Disciplina**: Selecione uma disciplina (filtrada por instituição)
   - **Período letivo**: Datas de início e término (opcional)
   - **Descrição**: Descrição opcional
3. Clique em **Criar**

![Formulário de grupo](/images/organization-setup/group-02-create-form.webp)

### Ações do grupo

Cada card possui ações rápidas:
- **Editar**: Modificar os dados do grupo
- **Alunos**: Gerenciar os alunos do grupo
- **Notas**: Visualizar a tabela de notas
- **Esquema**: Configurar a escala de avaliação
- **Arquivar**: Ocultar o grupo sem excluí-lo


### Grupos arquivados

Use o botão **Mostrar arquivados** para visualizar grupos de períodos anteriores. Você pode reativá-los a qualquer momento.

<Callout type="info">
Arquivar é melhor do que excluir: você mantém os dados e pode restaurar o grupo mais tarde.
</Callout>
