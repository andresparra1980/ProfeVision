## Ajouter des étudiants

Vous pouvez ajouter des étudiants manuellement ou en les important depuis Excel.

### Accéder aux étudiants

1. Sur la carte du groupe, cliquez sur l'icône **Étudiants**
2. Ou naviguez vers **Groupes** → sélectionnez un groupe → **Étudiants**

![Students list](/images/organization-setup/students-01-group-list.webp)

### Ajouter manuellement

1. Cliquez sur **Ajouter un étudiant**
2. Recherchez par nom ou par identification
3. Sélectionnez l'étudiant dans les résultats
4. Ou créez-en un nouveau avec :
   - **Prénoms** : Prénoms de l'étudiant
   - **Noms** : Noms de famille de l'étudiant
   - **Identification** : Numéro d'identifiant
   - **E-mail** : Adresse e-mail (optionnel)

![Add student](/images/organization-setup/students-02-add-manual.webp)

### Importer depuis Excel

1. Cliquez sur **Importer depuis Excel**
2. Configurez les options :
   - **Séparer les noms** : Colonnes distinctes pour les prénoms et les noms
   - **Inclure les e-mails** : Ajouter une colonne pour les e-mails
3. Téléchargez le modèle
4. Remplissez le modèle avec les données
5. Téléchargez le fichier vers la plateforme
6. Vérifiez l'aperçu
7. Cliquez sur **Importer**

![Import Excel](/images/organization-setup/students-03-bulk-import.webp)

### Retirer un étudiant

Cliquez sur le **X** à côté d'un étudiant pour le retirer du groupe.

<Callout type="info">
Les étudiants peuvent appartenir à plusieurs groupes. Le fait de les retirer d'un groupe ne les supprime pas du système.
</Callout>
