## Créer une matière

Une matière représente un cours ou une classe que vous enseignez.

### Accéder aux matières

1. Dans la barre latérale, cliquez sur **Matières**
2. Vous verrez vos matières s'afficher sous forme de cartes

![Liste des matières](/images/organization-setup/subject-01-list.webp)

### Ajouter une nouvelle matière

1. Cliquez sur le bouton **Nouvelle matière**
2. Remplissez le formulaire :
   - **Nom** : Nom de la matière (minimum 2 caractères)
   - **Description** : Description facultative
   - **Établissement** : Sélectionnez un établissement (facultatif)
3. Cliquez sur **Créer**

![Formulaire de matière](/images/organization-setup/subject-02-create-form.webp)

### Modifier ou supprimer

Chaque carte dispose de boutons pour :
- **Modifier** : Modifier le nom, la description ou l’établissement
- **Supprimer** : Retirer la matière et les données associées


<Callout type="warn">
La suppression d'une matière entraîne également la suppression de ses groupes, examens et résultats.
</Callout>
