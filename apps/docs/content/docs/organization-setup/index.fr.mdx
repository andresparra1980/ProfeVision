## Configuration de l'Organisation

Avant de créer des examens, vous devez configurer votre structure organisationnelle.

### Hiérarchie

ProfeVision utilise une hiérarchie à 3 niveaux :

1. **Institution** → Votre école ou université
2. **Matière** → Les cours que vous enseignez
3. **Groupe** → Vos classes d'étudiants
