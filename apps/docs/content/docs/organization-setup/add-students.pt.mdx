## Adicionar alunos

Você pode adicionar alunos manualmente ou importando do Excel.

### Acessar alunos

1. No cartão do grupo, clique no ícone **Alunos**
2. Ou navegue até **Grupos** → selecione um grupo → **Alunos**

![Students list](/images/organization-setup/students-01-group-list.webp)

### Adicionar manualmente

1. Clique em **Adicionar Aluno**
2. Pesquise pelo nome ou identificação
3. Selecione o aluno nos resultados
4. Ou crie um novo com:
   - **Nomes**: Primeiros nomes do aluno
   - **Sobrenomes**: Sobrenomes do aluno
   - **Identificação**: Número de ID
   - **E-mail**: Endereço de e-mail (opcional)

![Add student](/images/organization-setup/students-02-add-manual.webp)

### Importar do Excel

1. Clique em **Importar do Excel**
2. Configure as opções:
   - **Nomes separados**: Colunas separadas para nome e sobrenome
   - **Incluir e-mails**: Adicionar coluna de e-mail
3. Baixe o modelo
4. Preencha o modelo com os dados
5. Faça o upload do arquivo
6. Revise a pré-visualização
7. Clique em **Importar**

![Import Excel](/images/organization-setup/students-03-bulk-import.webp)

### Remover aluno

Clique no **X** ao lado de um aluno para removê-lo do grupo.

<Callout type="info">
Os alunos podem pertencer a vários grupos. Removê-los de um grupo não os exclui do sistema.
</Callout>
