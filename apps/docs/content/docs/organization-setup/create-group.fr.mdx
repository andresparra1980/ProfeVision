## Créer un groupe

Un groupe représente une classe ou une section d'étudiants.

### Accéder aux groupes

1. Dans la barre latérale, cliquez sur **Groupes**
2. Vos groupes s'afficheront sous forme de cartes

![Groups list](/images/organization-setup/group-01-list.webp)

### Ajouter un nouveau groupe

1. Cliquez sur le bouton **Nouveau Groupe**
2. Remplissez le formulaire :
   - **Nom** : Nom du groupe (ex: "3ème A")
   - **Établissement** : Sélectionnez un établissement
   - **Matière** : Sélectionnez une matière (filtrée par établissement)
   - **Période scolaire** : Dates de début et de fin (facultatif)
   - **Description** : Description facultative
3. Cliquez sur **Créer**

![Group form](/images/organization-setup/group-02-create-form.webp)

### Actions du groupe

Chaque carte dispose d'actions rapides :
- **Modifier** : Modifier les données du groupe
- **Étudiants** : Gérer les étudiants du groupe
- **Notes** : Consulter le tableau de notation
- **Barème** : Configurer l'échelle de notation
- **Archiver** : Masquer le groupe sans le supprimer


### Groupes archivés

Utilisez le bouton **Afficher les archives** pour voir les groupes des périodes précédentes. Vous pouvez les réactiver à tout moment.

<Callout type="info">
L'archivage est préférable à la suppression : vous conservez les données et pouvez restaurer le groupe plus tard.
</Callout>
