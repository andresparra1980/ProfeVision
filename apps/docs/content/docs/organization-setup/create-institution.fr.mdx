## Créer une institution

Une institution représente votre école, université ou centre de formation.

### Accéder aux institutions

1. Dans la barre latérale, cliquez sur **Institutions**
2. Vous verrez la liste de vos institutions (ou un état vide s'il s'agit de votre première)

![Institutions list](/images/organization-setup/institution-01-list.webp)

### Ajouter une nouvelle institution

1. Cliquez sur le bouton **New Institution**
2. Remplissez le formulaire :
   - **Nom** : Le nom de votre institution
   - **Type** : Université, Institut ou École
3. Cliquez sur **Create**

![Institution form](/images/organization-setup/institution-02-create-form.webp)

### Rechercher des institutions

Utilisez le champ de recherche pour filtrer par nom ou par type.


<Callout type="info">
Vous pouvez créer plusieurs institutions si vous enseignez dans différents établissements.
</Callout>
