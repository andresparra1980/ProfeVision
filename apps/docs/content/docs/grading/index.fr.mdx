## Notation par caméra

ProfeVision utilise la technologie OMR (Reconnaissance Optique de Marques) pour noter automatiquement les examens à l'aide de la caméra de votre téléphone.

### Fonctionnement

1. Les étudiants marquent leurs réponses sur des feuilles imprimées
2. Vous prenez une photo de chaque feuille avec votre téléphone
3. Le système détecte les réponses cochées
4. La note est calculée et enregistrée automatiquement
