## Vista general

El escaneo OMR permite calificar exámenes en segundos usando la cámara.

### Qué es OMR

**OMR** (Optical Mark Recognition) es tecnología que detecta marcas en papel. ProfeVision la usa para:

- Detectar el código QR del estudiante
- Leer las burbujas marcadas
- Calcular la calificación

### El proceso

```
Foto → Detección QR → Lectura de burbujas → Calificación → Guardado
```

![Proceso OMR](/images/grading/scan-01-answer-sheet.webp)

### Requisitos

| Elemento | Requisito |
|----------|-----------|
| Hoja de respuesta | Impresa correctamente |
| QR | Visible y sin daños |
| Burbujas | Marcadas con lápiz oscuro |
| Iluminación | Buena luz, sin sombras |

### Tiempo por hoja

- Captura: ~2 segundos
- Procesamiento: ~3 segundos
- **Total: ~5 segundos por estudiante**

<Callout type="info">
Con ProfeVision puedes calificar una clase de 30 estudiantes en menos de 3 minutos.
</Callout>
