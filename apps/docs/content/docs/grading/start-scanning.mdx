## Iniciar escaneo

Accede al escáner desde cualquier parte del dashboard.

### Botón flotante

El botón de escaneo aparece en la **esquina inferior derecha** de tu pantalla.

![Botón flotante](/images/grading/scan-02-fab-button.webp)

### Verificar límites

Antes de escanear, el sistema verifica tu cuota:

- Verás cuántos escaneos has usado
- Si alcanzas el límite, aparece una advertencia
- Usuarios Plus tienen escaneos ilimitados


### Pantalla de instrucciones

Al iniciar verás instrucciones con consejos:

1. Asegúrate de tener buena iluminación
2. Coloca la hoja sobre una superficie plana
3. Captura toda la hoja en el encuadre
4. Evita sombras y reflejos

![Instrucciones](/images/grading/scan-03-wizard-start.webp)

### Continuar

Haz clic en **Continuar** para abrir la cámara.

<Callout type="info">
El escáner funciona mejor con el teléfono en posición horizontal.
</Callout>
