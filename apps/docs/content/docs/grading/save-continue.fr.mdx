## Enregistrer et continuer

Après avoir vérifié, enregistrez les résultats et continuez avec d’autres copies.

### Enregistrer les résultats

1. Vérifiez que tout est correct
2. Cliquez sur **Enregistrer**
3. Attendez la confirmation


### Confirmation

Vous verrez un message de réussite lorsque les résultats seront enregistrés.

![Confirmation](/images/grading/scan-06-success.webp)

### Options après l'enregistrement

| Option | Action |
|--------|--------|
| Numériser un autre | Retourne à la capture d'image |
| Terminer | Ferme le numériseur |

### Numériser plusieurs copies

Pour numériser plusieurs copies à la suite :

1. Enregistrez la première copie
2. Cliquez sur **Numériser un autre**
3. Répétez le processus
4. Une fois terminé, cliquez sur **Terminer**

### Doublons

Si vous numérisez une copie qui a déjà été numérisée :

- Le système détecte le doublon
- Vous pouvez choisir de mettre à jour ou d'annuler
- Les résultats précédents sont remplacés si vous mettez à jour

<Callout type="info">
Le système enregistre l'image numérisée comme référence. Vous pouvez la consulter dans les résultats de l'examen.
</Callout>
