## Save and continue

After verifying, save the results and continue with more sheets.

### Save results

1. Check that everything is correct
2. Click **Save**
3. Wait for confirmation


### Confirmation

You'll see a success message when results are saved.

![Confirmation](/images/grading/scan-06-success.webp)

### Options after saving

| Option | Action |
|--------|--------|
| Scan another | Returns to image capture |
| Finish | Closes the scanner |

### Scanning multiple sheets

To scan several sheets in a row:

1. Save the first sheet
2. Click **Scan another**
3. Repeat the process
4. When done, click **Finish**

### Duplicates

If you scan a sheet that was already scanned:

- System detects the duplicate
- You can choose to update or cancel
- Previous results are replaced if you update

<Callout type="info">
The system saves the scanned image as reference. You can view it in the exam results.
</Callout>
