## Correção por Câmera

O ProfeVision utiliza a tecnologia OMR (Reconhecimento Óptico de Marcas) para corrigir exames automaticamente com a câmera do seu celular.

### Como funciona

1. Os alunos marcam suas respostas em folhas impressas
2. Você tira uma foto de cada folha com seu celular
3. O sistema detecta as respostas marcadas
4. A nota é calculada e salva automaticamente
