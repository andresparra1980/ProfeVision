## Capture d'image

Numérisez les feuilles de réponses avec l'appareil photo de votre appareil.

### Ouvrir l'appareil photo

En cliquant sur la zone de capture, l'appareil photo de votre appareil s'ouvre.

![Capture area](/images/grading/scan-04-camera-view.webp)

### Options de capture

| Option | Description |
|--------|-------------|
| Flash | Activer/désactiver le flash |
| N/B | Mode noir et blanc (meilleur contraste) |

### Comment numériser

1. Placez la feuille sur une surface plane
2. Assurez-vous que l'intégralité de la feuille est visible
3. Vérifiez que le QR code est complètement visible
4. Prenez la photo


### Aperçu

Après la capture :

1. Vérifiez que l'image est nette
2. Si elle n'est pas correcte, cliquez sur **Reprendre**
3. Si elle est correcte, cliquez sur **Continuer**


### Traitement

Après avoir confirmé l'image, le système traite automatiquement la feuille de réponses.

### Conseils

- Utilisez un bon éclairage naturel ou artificiel
- Évitez les ombres sur la feuille
- N'inclinez pas trop le téléphone
- Gardez le téléphone stable

<Callout type="warn">
Si l'image est floue ou si le QR code n'est pas lisible, la numérisation échouera.
</Callout>
