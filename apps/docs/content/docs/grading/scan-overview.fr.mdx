## Aperçu

La numérisation OMR permet de noter les examens en quelques secondes à l'aide de la caméra.

### Qu'est-ce que l'OMR ?

L'**OMR** (Reconnaissance Optique de Marques) est une technologie qui détecte les marques sur le papier. ProfeVision l'utilise pour :

- Détecter le code QR de l'étudiant
- Lire les bulles cochées
- Calculer la notation

### Le processus
```
Photo → QR Detection → Bubble Reading → Grading → Saving
```
![OMR process](/images/grading/scan-01-answer-sheet.webp)

### Prérequis

| Élément | Exigence |
|---------|-------------|
| Feuille de réponses | Imprimée correctement |
| QR | Visible et non endommagé |
| Bulles | Remplies avec un crayon foncé |
| Éclairage | Bonne lumière, sans zones d'ombre |

### Temps par feuille

- Capture : ~2 secondes
- Traitement : ~3 secondes
- **Total : ~5 secondes par étudiant**

<Callout type="info">
Avec ProfeVision, vous pouvez noter une classe de 30 étudiants en moins de 3 minutes.
</Callout>
