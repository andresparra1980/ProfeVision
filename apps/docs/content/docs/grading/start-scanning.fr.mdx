## Commencer la numérisation

Accédez au numériseur depuis n'importe quel endroit du tableau de bord.

### Bouton flottant

Le bouton de numérisation apparaît dans le **coin inférieur droit** de votre écran.

![Floating button](/images/grading/scan-02-fab-button.webp)

### Vérifier les limites

Avant de numériser, le système vérifie votre quota :

- Vous verrez combien de numérisations vous avez utilisées
- Si vous atteignez la limite, un avertissement s'affiche
- Les utilisateurs Plus bénéficient de numérisations illimitées

### Écran d'instructions

Au démarrage, vous verrez des instructions avec des conseils :

1. Assurez-vous d'avoir un bon éclairage
2. Placez la feuille sur une surface plane
3. Capturez l'intégralité de la feuille dans le cadre
4. Évitez les ombres et les reflets

![Instructions](/images/grading/scan-03-wizard-start.webp)

### Continuer

Cliquez sur **Continuer** pour ouvrir la caméra.

<Callout type="info">
Le numériseur fonctionne mieux avec le téléphone en position paysage.
</Callout>
