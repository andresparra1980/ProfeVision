## Overview

OMR scanning allows grading exams in seconds using the camera.

### What is OMR

**OMR** (Optical Mark Recognition) is technology that detects marks on paper. ProfeVision uses it to:

- Detect the student's QR code
- Read marked bubbles
- Calculate the grade

### The process

```
Photo → QR Detection → Bubble Reading → Grading → Saving
```

![OMR process](/images/grading/scan-01-answer-sheet.webp)

### Requirements

| Element | Requirement |
|---------|-------------|
| Answer sheet | Correctly printed |
| QR | Visible and undamaged |
| Bubbles | Marked with dark pencil |
| Lighting | Good light, no shadows |

### Time per sheet

- Capture: ~2 seconds
- Processing: ~3 seconds
- **Total: ~5 seconds per student**

<Callout type="info">
With ProfeVision you can grade a class of 30 students in less than 3 minutes.
