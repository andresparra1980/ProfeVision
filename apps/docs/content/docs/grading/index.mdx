## Calificación con cámara

ProfeVision usa tecnología OMR (Reconocimiento Óptico de Marcas) para calificar exámenes automáticamente con la cámara de tu teléfono.

### Cómo funciona

1. Los estudiantes marcan sus respuestas en hojas impresas
2. Tomas una foto de cada hoja con tu teléfono
3. El sistema detecta las respuestas marcadas
4. Se calcula y guarda la calificación automáticamente
