# Bem-vindo ao <span className="pv-logo"><span className="profe">Profe</span><span className="vision">Vision</span></span>

<span className="pv-logo"><span className="profe">Profe</span><span className="vision">Vision</span></span> é uma plataforma abrangente para professores que simplifica a criação, gestão e correção de exames.

## Principais recursos

- **Criação de exames**: Crie exames de múltipla escolha com IA
- **Digitalização OMR**: Corrija folhas de respostas com a câmera do seu celular
- **Gestão de turmas**: Organize seus alunos por grupos e períodos
- **Relatórios detalhados**: Visualize o desempenho de seus alunos
