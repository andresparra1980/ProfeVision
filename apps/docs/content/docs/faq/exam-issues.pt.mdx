## Problemas com exames

### Não consigo criar um exame

**Pré-requisitos:**
1. Ter pelo menos uma instituição
2. Ter pelo menos uma disciplina
3. Ter pelo menos uma turma

Crie estes elementos primeiro em **Configuração da Organização**.

### Não consigo editar um exame publicado

Exames publicados possuem edição limitada para proteger os resultados. Você pode:
- Ativar/desativar questões
- Alterar a resposta correta

### Posso duplicar um exame?

Atualmente não existe uma função de duplicar. Crie um novo exame ou utilize a IA.

### Como faço para excluir um exame?

1. Vá para a lista de exames
2. Apenas exames em **rascunho** podem ser excluídos
3. Clique em excluir e confirme

<Callout type="warn">
Exames publicados não podem ser excluídos se possuírem resultados.
</Callout>

### O PDF não é gerado

- Verifique se o exame possui questões
- O exame deve estar publicado
- Deve possuir pelo menos uma turma atribuída

### As questões não foram salvas

Seu progresso é salvo no navegador. Se você trocou de dispositivo ou limpou os dados, o progresso foi perdido. Complete o exame em uma única sessão.
