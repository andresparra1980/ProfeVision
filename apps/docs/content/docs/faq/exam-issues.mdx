## Problemas con exámenes

### No puedo crear un examen

**Requisitos previos:**
1. Tener al menos una institución
2. Tener al menos una materia
3. Tener al menos un grupo

Crea estos elementos primero en **Configuración organizacional**.

### No puedo editar un examen publicado

Los exámenes publicados tienen edición limitada para proteger los resultados. Puedes:
- Habilitar/deshabilitar preguntas
- Cambiar la respuesta correcta

### ¿Puedo duplicar un examen?

Actualmente no hay función de duplicar. Crea un nuevo examen o usa la IA.

### ¿Cómo elimino un examen?

1. Ve a la lista de exámenes
2. Solo exámenes en **borrador** se pueden eliminar
3. Haz clic en eliminar y confirma

<Callout type="warn">
Los exámenes publicados no se pueden eliminar si tienen resultados.
</Callout>

### El PDF no se genera

- Verifica que el examen tenga preguntas
- El examen debe estar publicado
- Debe tener al menos un grupo asignado

### Las preguntas no se guardaron

Tu progreso se guarda en el navegador. Si cambiaste de dispositivo o limpiaste datos, se perdió. Completa el examen en una sesión.
