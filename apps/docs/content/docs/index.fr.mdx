# Bienvenue sur <span className="pv-logo"><span className="profe">Profe</span><span className="vision">Vision</span></span>

<span className="pv-logo"><span className="profe">Profe</span><span className="vision">Vision</span></span> est une plateforme complète pour les enseignants qui simplifie la création, la gestion et la notation des examens.

## Fonctionnalités clés

- **Création d'examens** : Créez des examens à choix multiples grâce à l'IA
- **Numérisation OMR** : Notez les feuilles de réponses avec l'appareil photo de votre téléphone
- **Gestion de groupes** : Organisez vos étudiants par groupes et par périodes
- **Rapports détaillés** : Visualisez la performance de vos étudiants
