## Passer à la version Plus

Débloquez toutes les fonctionnalités sans limites.

### Depuis Mon forfait

1. Allez dans **Mon forfait** dans la barre latérale
2. Cliquez sur **Passer à la version Plus**
3. Vous serez redirigé vers la page de paiement


### Processus de paiement

1. Saisissez vos informations de paiement
2. Confirmez l'achat
3. Votre compte se met à jour automatiquement


### Moyens de paiement

Nous acceptons :
- Cartes de crédit/débit
- PayPal

### Activation

Après le paiement :
- Votre forfait passe immédiatement à la version Plus
- Les limites sont supprimées
- Vous recevez un e-mail de confirmation

### Facturation

- Prélèvement mensuel automatique
- Vous pouvez annuler à tout moment
- Factures disponibles dans le portail

Le paiement est traité en toute sécurité par Polar.
