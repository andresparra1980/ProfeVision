## Gérer l'abonnement

Gérez votre abonnement depuis le portail client.

### Accéder au portail

1. Allez dans **Mon forfait**
2. Cliquez sur **Gérer l'abonnement**
3. Le portail Polar s'ouvre


### Options disponibles

Dans le portail, vous pouvez :

| Option | Description |
|--------|-------------|
| Voir les factures | Historique des paiements |
| Mettre à jour le paiement | Modifier le mode de paiement |
| Résilier | Annuler l'abonnement |

### Résilier l'abonnement

1. Dans le portail, cliquez sur **Résilier**
2. Confirmez la résiliation
3. Vous conservez l'accès Plus jusqu'à la fin de la période

### Après la résiliation

- Accès Plus jusqu'à la fin de la période payée
- Ensuite, vous repassez au forfait gratuit
- Vos données sont conservées
- Vous pouvez réactiver l'abonnement à tout moment

<Callout type="info">
La résiliation ne supprime pas vos données. Seules les limites du forfait gratuit s'appliquent.
</Callout>
