# Welcome to ProfeVision

ProfeVision is a comprehensive platform for teachers that simplifies exam creation, management, and grading.

## Key features

- **Exam creation**: Create multiple choice exams with ease
- **OMR scanning**: Grade answer sheets automatically
- **Group management**: Organize your students by groups and periods
- **Detailed reports**: Visualize your students' performance
