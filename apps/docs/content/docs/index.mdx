# Bienvenido a <span className="pv-logo"><span className="profe">Profe</span><span className="vision">Vision</span></span>

<span className="pv-logo"><span className="profe">Profe</span><span className="vision">Vision</span></span> es una plataforma integral para profesores que simplifica la creación, administración y calificación de exámenes.

## Características principales

- **Creación de exámenes**: Crea exámenes de opción múltiple con IA
- **Escaneo OMR**: Califica hojas de respuestas con la cámara del celular
- **Gestión de grupos**: Organiza tus estudiantes por grupos y periodos
- **Reportes detallados**: Visualiza el rendimiento de tus estudiantes
