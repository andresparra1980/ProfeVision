# Bienvenido a ProfeVision

ProfeVision es una plataforma integral para profesores que simplifica la creacion, administracion y calificacion de examenes.

## Caracteristicas principales

- **Creacion de examenes**: Crea examenes de opcion multiple con facilidad
- **Escaneo OMR**: Califica hojas de respuestas automaticamente
- **Gestion de grupos**: Organiza tus estudiantes por grupos y periodos
- **Reportes detallados**: Visualiza el rendimiento de tus estudiantes
